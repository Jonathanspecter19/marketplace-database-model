# Modelo de Banco de Dados para Marketplace (MySQL)

Este projeto apresenta a modelagem e implementação de um banco de dados relacional para um marketplace multivendedor.

## Objetivo do Projeto

O objetivo deste projeto é simular um banco de dados capaz de gerenciar:

- Clientes
- Vendedores
- Produtos
- Pedidos
- Pagamentos
- Repasses para vendedores
